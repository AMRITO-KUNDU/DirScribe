# Contributing

Thanks for contributing to DirScribe.

## Setup

- Python 3.10+
- Install dependencies: `pip install -r requirements.txt`

## Run

- GUI: `python -m gui`
- CLI: `python -m cli.main --help`

## Guidelines

- Keep changes focused and small.
- Prefer reusing `core/` logic (don’t duplicate it in GUI/CLI).
- If you change behavior, update `README.md`.
